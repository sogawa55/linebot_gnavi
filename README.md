# Wifiと電源のある店舗検索 LINEbot

## botの特徴
* ぐるなびAPIで位置情報からお店検索が可能
* Wifiと電源が使えるランチから営業をしているお店を検索
* 半径500メートル以内

## 使用技術
* ruby 2.3
* ruby on rails 4.2.5
* Paas：heroku
* db：Postgresql
* lib：faraday
* LINE Messaging API
* ぐるなびレストラン検索 API
